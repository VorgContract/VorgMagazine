<h3 id="croc"><a href="https://github.com/schollz/croc">croc</a></h3>

<span><strong><i class="fa fa-star">Star 18.3k</i> <i class="fa fa-eye">Watch 230</i> <i class="fa fa-code-fork">Fork 802</i></strong>    </span>

`croc` is a tool that allows any two computers to simply and securely transfer files and folders. AFAIK, *croc* is the only CLI file-transfer tool that does **all** of the following:

- allows **any two computers** to transfer data (using a relay)
- provides **end-to-end encryption** (using PAKE)
- enables easy **cross-platform** transfers (Windows, Linux, Mac)
- allows **multiple file** transfers
- allows **resuming transfers** that are interrupted
- local server or port-forwarding **not needed**
- **ipv6-first** with ipv4 fallback
- can **use proxy**, like tor

<code>

\>\>\> from pottery import RedisSet
\>\>\> basket = RedisSet({'apple', 'orange', 'apple', 'pear', 'orange', 'banana'}, redis=redis, key='basket')
\>\>\> sorted(basket)
['apple', 'banana', 'orange', 'pear']
\>\>\> 'orange' in basket
True
\>\>\> 'crabgrass' in basket
False

\>\>\> a = RedisSet('abracadabra', redis=redis, key='magic')
\>\>\> b = set('alacazam')
\>\>\> sorted(a)
['a', 'b', 'c', 'd', 'r']
\>\>\> sorted(a - b)
['b', 'd', 'r']
\>\>\> sorted(a | b)
['a', 'b', 'c', 'd', 'l', 'm', 'r', 'z']
\>\>\> sorted(a & b)
['a', 'c']
\>\>\> sorted(a ^ b)
['b', 'd', 'l', 'm', 'r', 'z']
